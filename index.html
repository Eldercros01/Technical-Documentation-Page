Introduction

JavaScript is a versatile language used for both frontend and backend development.

It is interpreted, lightweight, and an essential part of the web.

console.log('Hello, World!');
  • Frontend Development
  • Backend Development
JavaScript Syntax

JavaScript syntax is the set of rules that define a correctly structured program.

Statements can be separated by semicolons.

let x = 10; const y = "hello";
  • Semicolons are optional
  • Case-sensitive
Variables

Variables are containers for storing data values.

You can declare variables using var, let, or const.

var name = "John"; let age = 30; const PI = 3.14;
  • Use let and const in modern JS
Functions

Functions are blocks of code designed to perform a particular task.

You can declare them using function or arrow syntax.

function greet() { return "Hi"; } const greet = () => "Hi";
  • Reusable code
  • Encapsulation
ES6 Features

ES6 introduced many new features to JavaScript.

Some examples include arrow functions, let/const, template literals, and destructuring.

const [a, b] = [1, 2]; let msg = `Hello ${name}`;
  • Arrow functions
  • Template literals
  • Default parameters
